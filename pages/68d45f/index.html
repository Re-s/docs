# 如何给 Deepin|UOS 反馈问题，报告缺陷，或提一些建议？

在 linuxdeepin (opens new window) 里对应仓库，正常提交 Issues 即可；此处是 Deepin 查看问题缺陷来源，也是你与其他人交流之地，尝试自行解决。

建议言简意赅，附详细环境，复现步骤即可，描述清楚即可，便于测试小姐姐复现问题；

推荐英文交流，关于 Issues （目前常用仓库）会自动 bot @ 相关人员。
